# MapLibre GL JS Example

A simple React application demonstrating MapLibre GL JS fundamentals, based on the official MapLibre documentation.

## Getting Started

Install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm run dev
```

Open your browser to the URL shown in the terminal.

## What's Inside

- **React 18** with Vite for fast development
- **MapLibre GL JS** - Open-source mapping library
- **Basic interactive map** with navigation controls
